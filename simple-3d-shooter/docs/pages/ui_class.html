# The UI Class The UI Class was created to provide a basic interface for [BabylonJS GUI](https://doc.babylonjs.com/how_to/gui). You can improve this class following the BabylonJS GUI conventions. We recommend using the official GUI methods to make your game UI instead of CSS/HTML because it can have an impact on the game performance. At this moment, this class offers the following methods: **addText** - Adds a Text to the UI instance. The options supported are the same of BabylonJS GUI Text **addButton** - Adds a Button to the UI instance. The options supported are the same of BabylonJS GUI Button **show** - Shows all the UI instance elements **hide** - Hides all the UI instance elements You can show/hide an especific element setting your isVisible property, as we made on **FirstLevel.js**: ``` this.hasMadeRecordTextControl.isVisible = false; ```
